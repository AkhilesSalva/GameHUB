## ⚙️ Cara Instalasi

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di lingkungan lokal Anda.

1. **Clone repository ini**
    ```bash
    git clone https://github.com/username-anda/game-hub.git
    ```

2. **Masuk ke direktori proyek**
    ```bash
    cd game-hub
    ```

3. **Install dependencies yang dibutuhkan** (sesuaikan dengan environment Anda)
    ```bash
    npm install
    # atau
    yarn install
    # atau
    composer install
    ```

4. **Setup file environment**
    Buat file `.env` dari contoh file `.env.example` dan sesuaikan konfigurasinya, terutama koneksi database:
    ```bash
    cp .env.example .env
    ```

5. **Jalankan aplikasi** (sesuaikan dengan framework Anda)
    ```bash
    npm run dev
    # atau
    php artisan serve
    ```

6. Buka browser dan akses:  
   **http://localhost:3000**
